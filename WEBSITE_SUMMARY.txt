## 🚀 Email Configuration (IMPORTANT!)

### To Enable Contact Form Emails:

1. **Open** `script.js` file in your editor
2. **Find** this line (around line 65):
   ```javascript
   const response = await fetch('https://formsubmit.co/your-email@gmail.com', {
   ```
3. **Replace** `your-email@gmail.com` with your actual email:
   ```javascript
   const response = await fetch('https://formsubmit.co/larry.bulus@gmail.com', {
   ```
4. **Save** the file
5. **Test** the form - on first submission, you'll get a verification email
6. **Verify** - Click the verification link
7. **Done!** All future submissions will arrive in your inbox

**Provider**: FormSubmit.co (Free, no registration needed)

## 🛠️ Customization Guide

### Change Colors
Edit `styles.css` variables at the top:
```css
:root {
    --primary-blue: #2563eb;  /* Change this */
    --primary-blue-light: #3b82f6;  /* And this */
    /* ... other colors ... */
}
```

### Update Text Content
Edit `index.html` to change:
- Service descriptions
- About section bio
- Form labels
- Footer text

### Change Hero Background
Edit `styles.css` `.hero` section:
```css
.hero {
    background: linear-gradient(135deg, #f0f9ff 0%, #e0f2fe 100%);
    /* Modify these colors */
}
```

### Add Your Photo
Replace or add images in the HTML sections.
